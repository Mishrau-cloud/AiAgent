# AI Specialization Agent – Overview

## What Is This Agent?

The **AI Specialization Agent** is a Microsoft Copilot Studio-based conversational agent designed to help partners quickly access relevant technical information, best practices, and guided support for AI-related topics.

Partners can interact with this agent to:

- Get instant answers to AI product and solution questions.
- Navigate Microsoft AI offerings and specialization requirements.
- Access curated knowledge from approved documentation, websites, and uploaded resources.
- Receive step-by-step guidance on implementation and enablement scenarios.

## Who Should Use This Agent?

This agent is intended for **Microsoft partners** who are:

- Pursuing or maintaining an AI specialization or advanced specialization.
- Onboarding to Microsoft AI products (Azure OpenAI, Copilot, etc.).
- Looking for a self-service reference tool during pre-sales, solutioning, or implementation phases.

## How to Use This Agent

1. Open the agent through the shared Copilot Studio link or embedded web channel provided by your organization.
2. Type your question or topic in natural language (e.g., *"What are the requirements for AI specialization?"* or *"How do I configure Azure OpenAI?"*).
3. The agent will respond using its configured knowledge base, which includes both website sources and uploaded PDF documents.

## Knowledge Base

The agent's knowledge base is kept up to date with:

| Source Type | Examples |
|-------------|----------|
| Website | Microsoft Learn, Partner Center docs, Azure documentation |
| PDF / Files | Specialization requirement guides, solution playbooks |
